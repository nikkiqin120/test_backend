# Backend QA Automation Demo (Python)

### Project Overview
This repository demonstrates a foundational automated testing framework for backend services, focusing on **API Validation**, **Business Logic**, and **Database Integrity**.

### Core Competencies Demonstrated:
* **API Automation:** Built using `Pytest` and `Requests`.
* **Testing Methodologies:** Includes functional testing and end-to-end (E2E) business workflows.
* **Database Validation:** Integration of SQL-based checks to ensure backend data consistency.
* **Clean Code:** Adheres to modular design for easy maintenance.

### How to Run:
1. Clone the repo.
2. Install dependencies: `pip install pytest requests`
3. Execute tests: `pytest -v -s`
